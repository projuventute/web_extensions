# Workspace Instructions

## Scope

This repository publishes independent browser snippets. Keep changes confined to the relevant top-level folder.

## Delivery

- Inspect the live script URL after a CDN-backed change. `@latest` can be stale until jsDelivr cache purge completes.
- Prefer immutable jsDelivr tag URLs for website embeds. A mutable `@latest` URL follows `main`.
- Do not change external widget APIs, CDN hosts, or initialization timing without checking supported browsers.
- Do not commit or push unless requested.

## RaiseNow

Read `raisenow/README.md` and `raisenow/AGENTS.md` before changing donation-widget files.
